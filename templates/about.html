About PhishGuard

A machine learning–powered URL phishing detection system.

What is PhishGuard?

PhishGuard analyses URLs in real time using a combination of TF-IDF character n-grams and hand-crafted lexical features to classify whether a link is safe or a potential phishing threat. Results include a confidence score and a breakdown of suspicious indicators.

Dataset

Source: Kaggle — Malicious URLs Dataset. The training set uses 10,000 balanced samples across four categories:

🟢 Benign URLs
🔴 Phishing URLs
🟠 Malware URLs
🟡 Defacement URLs

Machine Learning Pipeline

📝 TF-IDF (char n-grams 2–4)
🔬 Lexical Feature Extraction
⚖️ Standard Scaler
🧮 Logistic Regression
Model Accuracy
91.95%
Evaluated on 20% held-out test set
