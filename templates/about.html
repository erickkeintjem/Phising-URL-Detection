About Phishing Detection System

Project Description

Sistem ini digunakan untuk mendeteksi URL phishing menggunakan Machine Learning dan analisis fitur URL.

Dataset

Dataset berasal dari Kaggle dengan kategori:

  • Benign URL
  • Phishing URL
  • Malware URL
  • Defacement URL

Total data training yang digunakan: 10.000 URL

Machine Learning

  • TF-IDF Vectorizer
  • Lexical Feature Extraction
  • Logistic Regression

Model Performance

Accuracy: 91.95%

Technologies

  • Python
  • Flask
  • SQLite
  • HTML
